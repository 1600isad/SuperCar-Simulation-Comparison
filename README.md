# SuperCar-Simulation-Comparison-project
Streamlit website using python, pandas, and matplot to simulate realtime drag races and display car info and pictures and videos of each super carâ€”all pulled dynamically from a CSV dataset.

-----Features------

Car selection sidebar for quick navigation

Dynamic data loading from a CSV file using pandas

Car images displayed from external URLs (no local downloads required)

Embedded YouTube videos with autoplay support

Quarter-mile simulation (optional) using physics-based acceleration modeling

Clean, responsive UI powered by Streamlit

Easy to extend with more car data, images, and videos

Running the App Locally

# 1. Install dependencies:

pip install -r requirements.txt


# 2. Run Streamlit:

streamlit run Home.py
